# StoneLift

A hastily put together Workout App that runs 100% offline.

Unlike many Workout Logging Apps that restrict basic features behind paywalls or inundate users with ads, I've created a fitness tool that's tailored to my needs. It offers the essential features for crafting and tracking workouts without any limitations.
